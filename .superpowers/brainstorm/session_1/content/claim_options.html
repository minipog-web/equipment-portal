How should the 'Claim' feature work?

Select the option that best fits the desired workflow for Marano Eye Care Admin.

A

Inline Text Input

Clicking "⚙️ Claim" on a request row opens a small inline text input right inside the table row. The user types their name and hits "Confirm" to immediately set their claim.

B

Staff Dropdown Selection

Clicking "⚙️ Claim" opens a dropdown of common staff members (e.g., Adam, Veronica, Staff) plus an "Other" option. Simplifies claiming by removing manual typing for common users.

C

Modal Overlay Dialog

Clicking "⚙️ Claim" opens a centered modal dialog overlay, containing the equipment details, a technician name field, and an optional notes field (e.g. "Arriving in 10 minutes").
